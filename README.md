# 杨过同城 C 端预览

这是一个原生微信小程序 C 端预览项目，用本地模拟数据串联首页、现金券、置换、抽奖和个人中心等主要页面。当前版本面向开发预览和界面验证，不连接真实业务服务。

## 如何打开

1. 打开微信开发者工具。
2. 选择导入项目，并将项目目录指向本仓库根目录。
3. 使用本地调试模式预览页面；如需查看规则说明，可进入抽奖页后打开规则入口，或直接在页面列表中访问 `pages/rules/index`。

## 本地验证

在项目根目录运行：

```bash
node --test tests/*.test.js
```

该命令会执行页面结构、模拟服务和配置相关的 Node.js 自动化测试。

## 预览范围

- 使用模拟数据展示用户资料、现金券、券码、抽奖记录、置换商家权益和置换记录。
- 支持 C 端预览常见浏览路径，便于在微信开发者工具中检查页面文案、状态和跳转。
- 规则页说明抽奖参与、现金券使用、置换确认和开发预览免责声明。

## 不包含的生产能力

- 不包含真实支付。
- 不包含真实核销或真实验证。
- 不包含退款流程。
- 不包含后端 API、数据库、账号体系或商家后台。
