上下游链路说明

给开发和评审看的页面关系。

首页 → 优惠券

首页“买优惠券”和底部第二个 Tab 进入优惠券列表，列表承接团购券浏览、筛选与抢购。

优惠券 → 出示券码

用户购买或在“我的优惠券资产”中选择券后进入券码页，用于线下到店核销。

置换 → 发起置换 → 记录

置换页展示可兑换商家，提交置换申请后在记录页查看待确认、已完成、已失效状态。

抽奖 → 抽奖记录

抽奖页输入四位数参与，记录页展示参与号码、中奖号码和中奖权益。

我的 → 资产与记录

个人中心聚合优惠券资产、抽奖记录、置换记录、核销记录和资产流水入口。
